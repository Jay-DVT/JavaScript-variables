Introducción

JavaScript o JS, es un lenguaje de programación interpretado, este lenguaje es el principal motor detrás de la funcionalidad dentro de páginas web, esto se logra al juntarlo con los lenguajes de formato HTML y CSS. La capacidad de JavaScript de generar páginas web interactivas ha sido una de las partes fundamentales de la era moderna de la web. Alguna versión JavaScript se encuentra como motor dentro de todos los buscadores populares.

JavaScript es un lenguaje multiparadigma, admite estilos de programación imperativo, funcional y controlado por eventos.Cuenta con APIs para trabajar con textos, listas, días, expresiones regulares, y el Document Object Model (DOM). No cuenta con funciones para networking, almacenamiento, gráficos.

Tipos de Datos

  • Number: Representa números, ya sean enteros o de punto flotante. Ejemplo: let age = 25;
  • String: esenta secuencias de caracteres, usadas para texto. Ejemplo: let name = "Alice";
  • Boolean: Representa un valor lógico verdadero o falso. Ejemplo: let isActive = true;
  • Undefined: Indica que una variable ha sido declarada pero aún no se le ha asignado un valor. Ejemplo: let result;
  • Null: Representa la ausencia intencional de un valor de objeto. Ejemplo: let response = null;
  • Symbol: Introducido en ES6, representa un valor único que no es igual a ningún otro valor. Útil para propiedades de objeto únicas. Ejemplo: let sym = Symbol();
  • BigInt: bién introducido recientemente, para representar enteros muy grandes. Ejemplo: let bigInt = 1234567890123456789012345678901234567890n;

Objetos

Los objetos en JavaScript son colecciones de propiedades. Cada propiedad es un par de clave-valor. Los objetos pueden incluir funciones (llamadas métodos), arrays, incluso otros objetos.

let person = { name: "Alice", age: 25 };

Bucles

Los bucles son estructuras de control que repiten un bloque de código mientras se cumple una condición especificada.
  • for: Se usa para iterar sobre una secuencia, como un rango de números, un array, etc. 
    for (let i = 0; i < 5; i++) {
            console.log(i);
        }
  • while: Ejecuta un bloque de código mientras una condición especificada es verdadera. 
    let i = 0;
        while (i < 5) {
            console.log(i);
            i++;
        }
  • do-while: Similar al while, pero garantiza que el bloque de código se ejecute al menos una vez. 
    let i = 0;
        do {
            console.log(i);
            i++;
        } while (i < 5);
  • for-in: Se utiliza para iterar sobre todas las propiedades enumerables de un objeto. 
    let obj = {a: 1, b: 2, c: 3};
        for (let prop in obj) {
            console.log(prop + " = " + obj[prop]);
        }
  • for-of: Itera sobre los valores de un objeto iterable, como un array o string. 
    let arr = [10, 20, 30];
        for (let value of arr) {
            console.log(value);
        }

Asignación

La asignación en JavaScript es el proceso de asignar valores a variables usando el operador =.
  • Asignación Básica: Asigna valores con el operador =. 
    let x = 5;
  • Asignación Compuesta: Combina asignación con operaciones. 
    x += 10; // Equivalente a x = x + 10;
        x *= 3; // Equivalente a x = x * 3;

Variables

JavaScript se escribe dinámicamente. Esto significa que la misma variable puede contener valores de diferentes tipos.

  • var: La forma original de declarar una variable, antes de ES6 cuando ‘let’ y ‘const’ son introducidas, cuenta con un alcance de función, significando que sólo es accesible dentro de la función donde fue declarada, si fue declarada fuera de una función esta es global. la variable se puede utilizar antes de declararla debido a que es elevada, aunque el valor será undefined hasta el momento en que se declare la variable. Las variables pueden ser declaradas varias veces. 
    console.log(x); // undefined
        var x = 5;
  • let: Cuenta con alcance de bloque, esto apoya en el comportamiento de las variables dentro de bucles y condicionales. La variable declarada es elevada pero no inicializada, significando que retornara un ‘Reference Error’ si es llamada antes de ser declarada, declarar la misma variable dentro del mismo bloque no es permitido. 
    let x;
  • const: Cuento con alcance de bloque, la declaración de una variable con ‘const’ genera una referencia inmutable a una variable, significando que objetos si se pueden modificar. Requiere ser inicializada al ser declara y esta es elevada. 
    const z = 10;
        // z = 20; // Error
        const obj = { a: 1 };
        obj.a = 2; // Esto es posible

Usos

Functions

Las variables en funciones pueden ser locales (definidas dentro de la función) o globales (definidas fuera de cualquier función). Las variables locales sólo están accesibles dentro de la función donde se declaran. 

let a = "Global";
        function ejemploFuncion() {
            let b = "Local";
            return a + " y " + b;
        }
        console.log(ejemploFuncion()); // "Global y Local"
        // console.log(b); // Error: b no está definida fuera de la función

Bucles

Las variables son utilizadas para controlar la ejecución de bucles

for (let i = 0; i < 5; i++) {
            console.log(i);
        }

Objetos

En los objetos, las variables se utilizan para almacenar propiedades. Cada propiedad del objeto tiene un nombre (key) y un valor, que puede ser una variable. 

const persona = {
            nombre: "Gabriel",
            edad: 25
        };
        console.log(persona.nombre); // "Gabriel"
        persona.nombre = "Alberto";
        console.log(persona.nombre); // "Alberto"

Conclusión

JavaScript es un lenguaje de programación flexible que maneja una amplia gama de tipos de datos, desde los más básicos como números, cadenas de texto y booleanos hasta los más complejos como objetos. En cuanto a los bucles, ofrece estructuras como for, while, do-while, for-in y for-of para iterar y controlar de manera efectiva el flujo de ejecución del código. Finalmente, la asignación de valores a variables es un componente crucial, y hay operadores que permiten no solo la asignación directa, sino también la combinación de la asignación con operaciones aritméticas y lógicas para una codificación más sencilla.
